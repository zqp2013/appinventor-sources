App Inventor 2 组件参考文档

本文档描述您在使用App Inventor 2构建应用程序时所能用到的组件。

每个组件都可以有方法、事件和属性。这些属性都可以通过代码块获取或设置值，也有一些属性不能被程序改变（这些你只能通过代码块来获取值，而不能设置值）。 每个项目都带有附加文本注释，以指示项目的类型、是否是只读的以及是否只能存在于“界面设计”或“程序设计”中。
