访问MIT code服务器

网址：code.appinventor.mit.edu，正常情况下访问非常非常的慢，甚至大概率访问不了，有条件使用DL的话请使用DL。VIP会员可联系客服免费获取！

新建一个项目，拖出一个相应的组件

选中组件，在右侧的属性栏中查看“访问令牌” 或 “Token”属性，拷贝到中文网相应组件的“访问令牌” 或 “Token” 属性中，回车，保存项目即可。
