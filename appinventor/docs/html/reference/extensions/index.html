App Inventor 2 拓展参考文档

    本文档描述您在使用App Inventor 2构建应用程序时所能用到的拓展，以打造界面更加酷炫、功能更加强大的App。

    更多拓展请移步至《原版最全拓展一览》《中文版一览》。

      

  1. 【实用小技巧】从.aia项目文件中提取.aix拓展包
  2. 【ASCII编解码】AsciiConversion 拓展
  3. 【文件】FileTools 拓展：提供额外的更强大的文件相关操作
  4. 【图像】TaifunImage 拓展：图片压缩剪裁处理
  5. 【图像】SimpleBase64 拓展：图像Base64编解码传输
  6. 【图像】KIO4_Base64 拓展：更强大的图像Base64编解码工具，解码后的图像可写入到文件
  7. 【连接】MqttClient 拓展：轻量级物联网传输协议
  8. 【连接】ClientSocket 拓展：TCP/IP传输协议
  9. 【WIFI】TaifunWiFi 拓展：手机WiFi连接管理
  10. 【蓝牙】BlueToothLE 拓展：低功耗蓝牙
  11. 【数据库】SQLite 拓展：超流行兼容主流SQL的迷你本地数据库引擎
  12. 【数据库】TaifunSQLite 拓展：SQLite数据库引擎
  13. 【UI】NotificatioStyle 拓展：个性化通知栏样式
  14. 【UI】WordCloud 拓展：免费且酷炫的词云效果
  15. 【动态创建】DynamicComponents 拓展：动态创建AI2组件对象
  16. 【OCR】App Inventor 2 OCR 图片文字识别全方案总结
  17. 【FTP】App Inventor 2 FTP 上传下载全方案总结
  18. 【手机管理器】App Inventor 2 TaifunTM拓展：获取本机手机号码
