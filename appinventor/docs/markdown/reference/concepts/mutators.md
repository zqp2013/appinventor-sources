App Inventor 2 引入了一项新功能，允许某些块扩展、收缩甚至更改其功能。

![块扩展图标](images/mutatoricon.png)

顶部具有蓝色框白色齿轮的代码块都是可拓展块。

## 块扩展用法

块扩展可以改变形状，通过单击蓝色图标，用户可以将较小的块拖到较大的块中，从而改变原始块的形状和功能。
再次单击该图标将最小化拓展窗口，并显示修改后的块。
