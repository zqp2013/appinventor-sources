顶部的标题栏和状态栏通过 **设置Screen相关属性** 可以隐藏，图示如下：

![全屏设置](images/全屏设置.png)
![Screen属性](images/Screen属性.png)

而导航条是安卓手机的系统设置，默认是不启用不会出现的，如果启用了想隐藏也只能在手机设置中进行关闭操作。

通过上面的设置，App Inventor 2 就可以轻松实现全面屏效果了！！
