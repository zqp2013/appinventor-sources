## 预览PDF文档的方式

你可以使用`Activity启动器`查看已存储在你的设备上的 pdf 文档，也可以使用`Web客户端`通过网址URL打开 pdf 文档。

App Inventor 2 可以使用 .pdf 扩展名从程序包资产中查看 pdf 文件，不再需要外部 pdf 查看器！
