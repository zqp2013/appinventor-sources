# 获取有效“访问令牌”或“API密钥”的方式

## 新建一个项目，拖出一个相应的组件

选中组件，在右侧的属性栏中查看“访问令牌” 或 “Token”属性，拷贝到中文网相应组件的“访问令牌” 或 “Token” 属性中，回车，保存项目即可。
